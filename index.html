Pourquoi travailler avec moi ?

Possédant de nombreuse compétences en tant que développeur web fullstack JavaScript, je serais heureux de les mettre à profils dans la réalisation de votre projet.

Front End

Avec une maîtrise du framework React de JavaScript, je peux vous aider à concevoir toute la partie frontend de vos applications Web.

Back End

Je peux développer la partie backend de vos applications web en Node.js, avec les framework Express, Redux et MongoDB pour la gestion de base de données.

Mobile

Je maîtrise également le développement mobile, avec le framework React Native de JavaScript, qui s’adapte au plateforme IOS comme Android.
